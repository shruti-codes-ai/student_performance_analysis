Student Performance Analysis

Overview

This project analyzes student academic performance using Python, Pandas, and Matplotlib.

Features

- Load student data from a CSV file
- Calculate total marks
- Calculate average marks
- Find topper and lowest scorer
- Calculate subject-wise averages
- Identify best and weakest subjects
- Assign grades based on performance
- Analyze students above and below class average
- Visualize results using bar charts, histograms, and pie charts

Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
